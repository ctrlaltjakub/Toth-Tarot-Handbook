# Toth-Tarot-Handbook
An interactive, open-source encyclopedia of the Thoth Tarot system. Explores all 78 cards, the Kabbalistic Tree of Life, and the astrological framework with full correspondences between all three systems. Includes Jungian psychological interpretations, a cross-linked glossary, and interactive SVG diagrams. Built with React, TypeScript, and Vite.
